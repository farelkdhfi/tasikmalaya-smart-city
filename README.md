**Tasikmalaya Smart City** is a futuristic, fully responsive Command Center simulation designed to monitor urban operations. It integrates real-time data for traffic, health, safety, and environment into a unified, interactive dashboard interface.

> üèÜ **Award Winning Project.** This Smart City simulation dashboard was awarded **2nd Runner Up (Juara Harapan 2)** at the prestigious **FIT Competition 2024**, recognizing its advanced frontend interactivity and data visualization capabilities.

## ‚ú® Key Features

### üåç City Pulse & Environment
* **Real-time Monitoring:** Live tracking of Population Density, Air Quality Index (AQI), and Weather conditions.
* **Smart Waste:** IoT-enabled bin monitoring with auto-dispatch logic for collection trucks.
* **Sustainability:** Renewable energy mix visualization and carbon footprint tracking.

### üõ°Ô∏è Urban Safety & Security
* **Tactical Map:** Interactive zoning map with live incident tracking (Fire, Crime, Accidents).
* **Emergency Broadcast:** System-wide "Code Red" mode that overrides the UI for critical alerts.
* **CCTV Integration:** Simulated live camera feeds for traffic and security monitoring.

### üè• Smart Health System
* **Capacity Management:** Live monitoring of hospital bed occupancy and blood bank inventory.
* **Disease Control:** Tracking of outbreak clusters and vaccination rates.
* **Telemedicine:** Interface for booking appointments and AI-powered symptom checking.

### ‚ö° Engineering & UX
* **Interactive Data:** Rich visualization using **Recharts** for complex datasets (Trends, Demographics).
* **Immersive UI:** 3D elements powered by **Three.js** and fluid animations using **Framer Motion**.
* **Responsive Design:** Optimized layout that adapts seamlessly from desktop command centers to mobile devices.

## üöÄ Getting Started

Follow these steps to run the dashboard locally.

### Prerequisites

* Node.js (v16 or higher)
* npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/farelkdhfi/tasikmalaya-smart-city.git](https://github.com/farelkdhfi/tasikmalaya-smart-city.git)
    cd tasikmalaya-smart-city
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  **Open in browser**
    Navigate to `http://localhost:5173` (or the port shown in your terminal).
